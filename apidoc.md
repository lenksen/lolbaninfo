# LOL免密查封号API 开发文档

## 1. 概述
本API用于查询英雄联盟（LOL）账号的封禁状态信息，支持通过QQ号查询账号是否被封禁及封禁详情。

## 2. 基本信息
- **接口名称**：LOL免密查封号API
- **请求地址**：`https://yun.4png.com/api/query.html`
- **请求方式**：GET

## 3. 请求参数

| 参数名 | 类型   | 必填 | 说明         |
|--------|--------|------|--------------|
| qq     | string | 是   | 要查询的QQ号 |
| token  | string | 是   | 认证令牌，需通过网站注册登录后获取 |

## 4. Token获取方式
1. 访问网站：https://yun.4png.com
2. 注册账号并完成登录
3. 在API管理页面获取个人专属token
4. 每个用户的token不同，且具有时效性，请妥善保管

## 5. 请求示例
```
https://yun.4png.com/api/query.html?qq=664766099&token=11bdb3b5e49c31b0
```

## 6. 响应参数

### 6.1 通用响应结构
| 字段  | 类型   | 说明                           |
|-------|--------|--------------------------------|
| code  | int    | 状态码（200表示成功，400表示错误） |
| name  | string | 查询的QQ号                     |
| by    | string | 数据来源                       |
| data  | object | 返回数据对象（仅成功时返回）   |
| msg   | string | 返回消息                       |

### 6.2 data对象结构（仅当code=200时存在）
| 字段        | 类型   | 说明               |
|-------------|--------|--------------------|
| banmsg      | string | 封禁详细信息       |
| rammsg      | string | 剩余封禁时间       |
| return      | string | 封禁状态           |

## 7. 响应示例

### 7.1 成功响应示例
```json
{
    "code": 200,
    "name": "97077000",
    "by": "zai1.com",
    "data": {
        "banmsg": "账号97077000封禁至 2031-1-2 14:03:37，距离解封还有 1987天2小时32分41秒",
        "rammsg": "距离解封还有 1987天2小时32分41秒",
        "return": "封禁"
    },
    "msg": "此账号已被封禁"
}
```

### 7.2 错误响应示例
```json
{
    "code": 400,
    "msg": "参数缺失"
}
```

### 7.3 其他常见错误响应
```json
{
    "code": 401,
    "msg": "token无效或已过期"
}
```

```json
{
    "code": 404,
    "msg": "未找到相关账号信息"
}
```

## 8. 状态码说明

| 状态码 | 说明                 | 处理建议                     |
|--------|----------------------|------------------------------|
| 200    | 调用成功             | 正常处理返回数据             |
| 400    | 参数缺失或格式错误   | 检查请求参数是否完整且正确   |
| 401    | token无效或已过期    | 重新登录获取有效token        |
| 404    | 未找到相关账号信息   | 确认QQ号是否正确             |
| 500    | 服务器内部错误       | 稍后重试或联系技术支持       |
